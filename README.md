# 🚀 AI Task Tracker

A simple and modern task management web application built with HTML, CSS, and JavaScript.

## 📌 Overview

AI Task Tracker helps users manage their daily tasks efficiently with a clean interface and dark/light theme support.

## ✨ Features

- ✅ Add new tasks
- ✅ Mark tasks as completed
- ✅ Delete tasks
- ✅ Filter tasks (All / Active / Completed)
- ✅ Clear completed tasks
- ✅ Dark / Light mode toggle
- ✅ Data saved in LocalStorage
- ✅ Responsive design
- ✅ Smooth animations

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API

## 📂 Project Structure

```
/taskapp
│── index.html      # Main HTML structure
│── style.css       # Styles with dark/light theme support
│── script.js       # Application logic & LocalStorage handling
│── README.md       # Project documentation
```

## 🚀 Getting Started

1. Clone or download the project.
2. Open `index.html` in any modern browser.
3. Start adding your tasks!

## 🖥️ Usage

| Action          | How                                         |
| --------------- | ------------------------------------------- |
| Add a task      | Type in the input field and click **Add**   |
| Complete a task | Click the checkbox on the left              |
| Delete a task   | Hover over a task and click **✕**           |
| Filter tasks    | Click **All**, **Active**, or **Completed** |
| Clear completed | Click **Clear completed** in the footer     |
| Toggle theme    | Click the 🌙 / ☀️ button in the header      |
